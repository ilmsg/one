<li><strong>ผู้ชายนั้นเรียบง่าย:</strong> ฮาร์วีย์ยืนยันว่าผู้ชายส่วนใหญ่คิดคล้ายๆ กัน และสิ่งที่ขับเคลื่อนพวกเขาหลักๆ มี 3 อย่างคือ: <strong>ตัวตนของเขา</strong> (ตำแหน่งหน้าที่), <strong>สิ่งที่เขาทำ</strong> (อาชีพการงาน), และ <strong>สิ่งที่เขาได้รับ</strong> (รายได้) ผู้ชายจะรู้สึกเติมเต็มเมื่อบรรลุเป้าหมายใน 3 ด้านนี้ และก่อนที่เขาจะทำสำเร็จ ผู้หญิงและความสัมพันธ์มักจะเป็นเรื่องรอง</li>
<li><strong>ความรักของผู้ชายไม่เหมือนผู้หญิง:</strong> ผู้หญิงมักแสดงความรักด้วยคำพูดและการแสดงออกบ่อยครั้ง แต่ผู้ชายแสดงความรักผ่าน 3 การกระทำหลัก (3 Ps):
<ul>
<li><strong>Profess (ประกาศ):</strong> เขาจะแนะนำคุณในฐานะคนรักอย่างเป็นทางการกับคนรอบข้าง (เพื่อน, ครอบครัว)</li>
<li><strong>Provide (ดูแล):</strong> เขาจะพยายามดูแลคุณ ไม่ใช่แค่เรื่องเงิน แต่รวมถึงการช่วยเหลือ แก้ปัญหา ทำให้คุณรู้สึกมั่นคง</li>
<li><strong>Protect (ปกป้อง):</strong> เขาจะปกป้องคุณทั้งทางร่างกายและชื่อเสียงเมื่อมีคนมาทำร้ายหรือดูหมิ่น</li>
</ul>
</li>
</ul>
</li>
<li>
<p><strong>สิ่งที่ผู้ชายต้องการจากผู้หญิง:</strong></p>
<ul>
<li><strong>การสนับสนุน (Support):</strong> ผู้ชายต้องการผู้หญิงที่คอยให้กำลังใจ ชื่นชมในสิ่งที่เขาทำ และเป็นที่พักพิงทางใจ</li>
<li><strong>ความภักดี (Loyalty):</strong> เขาต้องการความมั่นใจว่าคุณจะอยู่เคียงข้างเขาเสมอ ไม่ว่าสถานการณ์จะเป็นอย่างไร</li>
<li><strong>เซ็กส์ (The Cookie):</strong> ฮาร์วีย์ใช้คำว่า "The Cookie" แทนเซ็กส์ และมองว่าเป็นความต้องการพื้นฐานที่สำคัญของผู้ชาย</li>
</ul>
</li>
<li>
<p><strong>ทำไมผู้ชายถึงทำบางอย่าง:</strong></p>
<ul>
<li><strong>เป้าหมายเมื่อแรกเจอ:</strong> เมื่อผู้ชายเข้ามาคุยกับคุณ เขามักมี 2 คำถามในใจ: คุณจะยอมนอนกับเขาไหม และต้องใช้ความพยายามแค่ไหน</li>
<li><strong>การแบ่งประเภทผู้หญิง:</strong> ผู้ชายมักแบ่งผู้หญิงเป็น 2 ประเภท คือ "Sports Fish" (คบเล่นๆ ชั่วคราว) กับ "Keepers" (คนที่อยากคบจริงจัง สร้างอนาคตด้วย) ฮาร์วีย์เน้นว่า ผู้หญิงเป็นคนกำหนดเองว่าจะอยู่ในประเภทไหน ผ่านการวางตัว การตั้งมาตรฐาน และการเรียกร้องความเคารพ</li>
<li><strong>ทำไมผู้ชายถึงนอกใจ:</strong> ฮาร์วีย์อธิบายว่าผู้ชายสามารถมีเซ็กส์โดยไม่มีอารมณ์ผูกพันได้ และอาจนอกใจหากความต้องการพื้นฐาน (การสนับสนุน, ความภักดี, เซ็กส์) ไม่ได้รับการตอบสนอง</li>
</ul>
</li>
<li>
<p><strong>กลยุทธ์สำหรับผู้หญิง (The Playbook):</strong></p>
<ul>
<li><strong>ตั้งมาตรฐาน (Set Standards):</strong> ผู้หญิงควรกำหนดความต้องการและความคาดหวังของตัวเองให้ชัดเจนตั้งแต่เนิ่นๆ และไม่ยอมลดมาตรฐานเพื่อใคร ผู้ชายจะเคารพผู้หญิงที่เคารพตัวเอง</li>
<li><strong>กฎ 90 วัน (The 90-Day Rule):</strong> ฮาร์วีย์แนะนำให้ผู้หญิงรออย่างน้อย 90 วันก่อนที่จะมีเพศสัมพันธ์กับผู้ชาย เพื่อให้เวลาในการทำความรู้จักนิสัยใจคอ และดูว่าผู้ชายคนนั้นจริงจังแค่ไหน</li>
<li><strong>5 คำถามสำคัญที่ผู้หญิงควรถาม:</strong> เพื่อทำความเข้าใจเป้าหมายและทัศนคติของผู้ชายเกี่ยวกับความสัมพันธ์และอนาคต เช่น เป้าหมายระยะสั้น/ยาวคืออะไร, มองความสัมพันธ์อย่างไร, คิดอย่างไรกับคุณ เป็นต้น</li>
<li><strong>การรับมือกับผู้ชายประเภทต่างๆ:</strong> เช่น "Mama's Boys" (ลูกแหง่ติดแม่) และวิธีสังเกตสัญญาณว่าผู้ชายคนนั้นจริงจังหรือไม่ (เช่น ยอมจ่ายค่าเดท, รักษาคำพูด, แนะนำให้รู้จักเพื่อน/ครอบครัว)</li>
<li><strong>ผู้หญิงเก่งและแกร่ง (แต่อาจจะเหงา):</strong> ฮาร์วีย์ให้คำแนะนำสำหรับผู้หญิงที่ประสบความสำเร็จและเป็นอิสระ แต่ยังหาความสัมพันธ์ที่มั่นคงไม่ได้</li>
<li><strong>ทำอย่างไรให้ได้แหวน:</strong> แนะนำวิธีเปลี่ยนความสัมพันธ์ที่คบกันอยู่ให้ก้าวไปสู่การแต่งงาน โดยการยืนหยัดเรื่องการกำหนดวันแต่งงาน</li>
</ul>
</li>
</ol>
<p><strong>โดยรวมแล้ว หนังสือเล่มนี้พยายามให้มุมมองจากฝั่งผู้ชาย เพื่อให้ผู้หญิงเข้าใจธรรมชาติ ความคิด และแรงจูงใจของผู้ชายมากขึ้น ซึ่งจะช่วยให้ผู้หญิงสามารถตัดสินใจและควบคุมทิศทางความสัมพันธ์ของตนเองได้ดีขึ้น โดยยังคงรักษาความเป็นผู้หญิง (Act Like a Lady) แต่มีความเข้าใจในมุมมองของผู้ชาย (Think Like a Man)</strong></p>
