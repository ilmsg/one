<p>วิธีอ่านคนในห้อง (How to Read Any Room) เน้นการสังเกตความเบี่ยงเบนในพฤติกรรมที่บ่งบอกถึงความสบายใจ (Comfort) หรือความไม่สบายใจ (Discomfort) โดยได้รับแรงบันดาลใจจาก โจ นาวาร์โร (Joe Navarro) อดีตเจ้าหน้าที่ FBI:</p>
<p>ภาพนี้แนะนำว่า การอ่านภาษากายเป็นทักษะที่สำคัญ โดยแบ่งการสังเกตออกเป็น 5 ส่วนหลักๆ ดังนี้:</p>
<ol>
<li>
<p><strong>ดวงตา (Eyes)</strong></p>
<ul>
<li><strong>ความสบายใจ:</strong>
<ul>
<li><strong>สบตา (Making eye contact):</strong> การมองตาคู่สนทนาเป็นสัญญาณของความมั่นใจและความเปิดเผย</li>
<li><strong>ยิ้มด้วยตา (Smiling with one's eyes):</strong> รอยยิ้มที่จริงใจมักจะแสดงออกทางดวงตาด้วย ทำให้รอบดวงตาดูหยีลงเล็กน้อย</li>
</ul>
</li>
<li><strong>ความไม่สบายใจ:</strong>
<ul>
<li><strong>การปิดกั้นสายตา (Eye-blocking):</strong> พฤติกรรม เช่น การหรี่ตา การปิดตา หรือการใช้มือบัง/ป้องดวงตา อาจบ่งบอกถึงความรู้สึกไม่สบายใจ ไม่เห็นด้วย หรือพยายามประมวลผลข้อมูลที่ไม่น่าพอใจ</li>
</ul>
</li>
</ul>
</li>
<li>
<p><strong>มือ (Hands)</strong></p>
<ul>
<li><strong>ความสบายใจ:</strong>
<ul>
<li><strong>ใช้มือประกอบการพูด (Speaking with hands):</strong> การใช้ท่าทางมืออย่างเหมาะสมขณะพูด ช่วยเสริมการสื่อสารและแสดงความกระตือรือร้น</li>
<li><strong>ท่ามือทรง " steeple" (ปลายนิ้วประกบกัน):</strong> การนำปลายนิ้วของทั้งสองมือมาแตะกันเป็นรูปคล้ายหลังคาโบสถ์ มักแสดงถึงความมั่นใจและความคิดที่ชัดเจน</li>
</ul>
</li>
<li><strong>ความไม่สบายใจ:</strong>
<ul>
<li><strong>บีบมือ/ขยำมือ (Handwringing):</strong> การนำมือมาบีบหรือขยำกันไปมา เป็นสัญญาณของความกังวล ความเครียด หรือความไม่สบายใจ</li>
<li><strong>ซ่อนมือ (Hiding hands):</strong> การเก็บมือไว้ด้านหลัง ซุกในกระเป๋า หรือซ่อนไว้ใต้โต๊ะ อาจบ่งบอกถึงความไม่มั่นใจ ความประหม่า หรือในบางกรณีอาจเป็นสัญญาณของการพยายามปิดบังบางสิ่ง (อาจเป็นการหลอกลวง)</li>
</ul>
</li>
</ul>
</li>
<li>
<p><strong>ริมฝีปาก (Lips)</strong></p>
<ul>
<li><strong>ความสบายใจ:</strong>
<ul>
<li><strong>ยิ้มอย่างอ่อนโยน (Smiling gently):</strong> รอยยิ้มที่ดูผ่อนคลาย เป็นธรรมชาติ</li>
<li><strong>ริมฝีปากผ่อนคลาย (Lips relaxed):</strong> ริมฝีปากอยู่ในลักษณะปกติ ไม่เกร็ง</li>
</ul>
</li>
<li><strong>ความไม่สบายใจ:</strong>
<ul>
<li><strong>เม้มปาก (Pursed together):</strong> การเม้มริมฝีปากแน่น อาจเป็นสัญญาณของความไม่เห็นด้วย ความโกรธที่เก็บกด หรือความเครียด</li>
<li><strong>ริมฝีปากหายไป (Disappearing):</strong> การเม้มปากจนแทบมองไม่เห็นริมฝีปาก เป็นสัญญาณที่ชัดเจนยิ่งขึ้นของความรู้สึกเชิงลบ</li>
</ul>
</li>
</ul>
</li>
<li>
<p><strong>เท้า (Feet)</strong></p>
<ul>
<li><strong>ความสบายใจ:</strong>
<ul>
<li><strong>เท้านิ่ง (Feet still):</strong> เท้าอยู่นิ่ง ไม่ขยับไปมามากนัก</li>
<li><strong>ชี้ไปทางคู่สนทนา (Pointing towards someone):</strong> โดยไม่รู้ตัว เท้าของเรามักจะชี้ไปในทิศทางที่เราสนใจหรืออยากจะไป</li>
<li><strong>ไขว่ห้าง (Crossed legs):</strong> ในบริบทที่ผ่อนคลาย การนั่งไขว่ห้างอาจเป็นสัญญาณของความสบายใจ (แต่ต้องดูบริบทอื่นประกอบ)</li>
</ul>
</li>
<li><strong>ความไม่สบายใจ:</strong>
<ul>
<li><strong>เท้าขยับมาก (Feet moving a lot):</strong> การขยับเท้าไปมา กระดิกเท้า หรืออยู่ไม่สุข บ่งบอกถึงความประหม่า ความกังวล หรือความต้องการที่จะออกจากสถานการณ์นั้น</li>
<li><strong>ชี้ออกไปทางอื่น (Pointing away):</strong> เท้าที่ชี้ไปทางประตูหรือทางออก อาจหมายความว่าคนนั้นอยากจะไปจากตรงนั้นแล้ว</li>
<li><strong>เคาะหรือเขย่าเท้า (Tapping or bouncing):</strong> เป็นอาการที่แสดงถึงความเบื่อหน่าย ความอดทนต่ำ หรือความกังวล</li>
</ul>
</li>
</ul>
</li>
<li>
<p><strong>ท่าทาง (Posture)</strong></p>
<ul>
<li><strong>ความสบายใจ:</strong>
<ul>
<li><strong>ยืน/นั่งตัวตรงผ่อนคลาย (Standing straight relaxed):</strong> ท่าทางที่ดูมั่นคงแต่ไม่เกร็ง แสดงถึงความมั่นใจและสบายใจ</li>
<li><strong>หันหน้าเข้าหาคู่สนทนา (Facing someone):</strong> การหันลำตัวเข้าหาคนที่คุยด้วย แสดงถึงความใส่ใจและความเปิดกว้าง</li>
<li><strong>โน้มตัวเข้าหา (Leaning in):</strong> การโน้มตัวเข้าหาเล็กน้อย แสดงถึงความสนใจในสิ่งที่อีกฝ่ายพูด</li>
</ul>
</li>
<li><strong>ความไม่สบายใจ:</strong>
<ul>
<li><strong>ห่อไหล่/หลังค่อม (Slouching):</strong> ท่าทางที่ดูหมดแรงหรือห่อตัว อาจแสดงถึงความไม่มั่นใจ ความเหนื่อยหน่าย หรือความรู้สึกพ่ายแพ้</li>
<li><strong>หันลำตัวหนี (Turning body away):</strong> การบิดหรือหันลำตัวออกจากคู่สนทนา เป็นสัญญาณของการปิดกั้น ไม่สนใจ หรืออยากออกจากวงสนทนา</li>
<li><strong>เอนตัวหนี (Leaning away):</strong> การเอนตัวออกจากคู่สนทนา แสดงถึงความไม่เห็นด้วย ความไม่สบายใจ หรือการสร้างระยะห่าง</li>
</ul>
</li>
</ul>
</li>
</ol>
<p>ภาพนี้สรุปว่า การสังเกตสัญญาณเหล่านี้จะช่วยให้เข้าใจความรู้สึกที่แท้จริงของคนรอบข้างได้ดีขึ้น ซึ่งเป็นทักษะที่มีประโยชน์อย่างยิ่งในการสื่อสารและสร้างความสัมพันธ์ และปิดท้ายด้วยชื่อ Ben Meer ซึ่งน่าจะเป็นผู้สร้างสรรค์อินโฟกราฟิกนี้ครับ</p>
